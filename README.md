# mrak.github.io

The `main` branch is for development, the `publish` branch is live.

## Developing

```
make build
```

## Development dependencies

Assumes a POSIX/UNIX system, with general CLI utilities such as `awk`, `sed`,
etc.

* `pandoc`
* `sassc`
* `awk`
* `make`
* `envsubst` (from `gettext`)
